Question-1: What are the different ways to select an element in the DOM?

Ans: Different ways to select an element in the DOM:

  1. By ID
  2. By Class
  3. By Tag Name
  4. By CSS Selector (querySelector)
  5. By CSS Selector (querySelectorAll)
  6. By Name Attribute

Question-2: What is the difference between innerHTML, innerText, and textContent?

Ans: The differences between innerHTML, innerText, and textContent:

  • innerHTML returns or sets the entire content, including HTML tags.
  • innerText returns or sets only the visible text, ignoring hidden elements.

Question-3: What is event delegation in the DOM?

Ans: Event Delegation is a design pattern where an event listener is attached to a parent element and manages event handling for child elements. It works based on the Event Bubbling mechanism.

Question-4: What is event bubbling in the DOM?

Ans: Event Bubbling is an event propagation mechanism where an event triggered on a child element propagates up to its parent elements. It works in a bottom-to-top manner.

Question-5: How do you create, add, and remove elements using JavaScript?

Ans: In JavaScript, we can create, add, and remove elements in the DOM dynamically using methods like createElement(), appendChild(), and removeChild().
